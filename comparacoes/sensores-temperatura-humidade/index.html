Aqara — melhor bateria

O SwitchBot tem a melhor precisão e o ecrã LCD integrado é muito prático — dá para ver a temperatura sem abrir o telemóvel. O Aqara é o rei da autonomia (2 anos com uma pilha CR2032). O Govee é o mais barato e não precisa de hub.

Aqara — Prós
- Fiabilidade excelente (nunca falha) - Bateria de 2 anos - Zigbee — rede mesh, não sobrecarrega o WiFi - Integração perfeita com Home Assistant e Xiaomi - Pequeno e discreto
Aqara — Contras
- Precisa de hub Zigbee (ou gateway Aqara) - Sem display (só na app) - Configuração inicial requer algum conhecimento técnico
Resumo
**Aqara** é o melhor para fiabilidade e integração com Home Assistant/ecossistema Xiaomi. Se já tem um hub Zigbee, não há melhor escolha. **SwitchBot** é o melhor para usar no dia-a-dia — o display integrado é super prático para ver a temperatura de relance. **Govee** é o melhor para iniciantes: barato, WiFi direto, sem complicações. Recomendações: - 1 sensor por divisão principal (quarto, sala, cozinha) - 1 sensor no exterior (varanda/jardim) se o modelo aguentar - Ligar os sensores a automações: desligar aquecedor acima de 22°C, ligar desumidificador acima de 70%

Perguntas Frequentes

Qual a diferença entre sensor Zigbee e WiFi?
Sensores Zigbee (como o Aqara) precisam de um hub mas consomem menos bateria (duram ~2 anos). Sensores WiFi (como o Govee) ligam-se diretamente ao router mas gastam mais bateria e são menos estáveis com muitos dispositivos.
O sensor funciona com Home Assistant?
Sim. O Aqara é o melhor para Home Assistant (via Zigbee2MQTT ou ZHA). O SwitchBot funciona com o hub deles. O Govee funciona com a integração oficial via cloud.
Quanto custa manter um sensor?
Praticamente nada. A bateria do Aqara dura 2 anos (custa ~1€ para substituir). O Govee usa 2 pilhas AAA e dura ~6-12 meses. O SwitchBot usa 1 CR2450 e dura ~6 meses.
